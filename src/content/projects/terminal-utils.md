This project started as a set of shell scripts that I eventually rewrote in Go for better performance and cross-platform compatibility.

### Key Features

- **p-log**: A high-speed log parser that can filter and format JSON logs in real-time.
- **git-audit**: A tool to quickly scan your local repositories for uncommitted changes or unpushed branches.

Building these tools helped me refine my understanding of Unix pipes and terminal escape sequences. It's now part of my daily workflow.
